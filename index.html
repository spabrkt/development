How to configure a file using the Arduino IDE to create a web server

The methods used here are related to the Async method rather than http

ESP32 Devkit 1C

PROJECT HARDWARE AND SET-UP

Contact details


Hardware

  • Breadboard PSU
  • Blue led which will be controlled with the web Server
  • Push button - also used with the web server
  • Green led - controlled in its own task
  • 3 x 10k resistors for the above. All items pulldown

Connections

  • Blue led as output from D5
  • Green led to output from D2
  • Push button as input on D17

Methods

  1. Create and save a blank reset file
  2. Set up and test basic led and pb without web server
  3. Add the following libraries
    • WiFi.h
    • AsyncTCP.h
    • ESPAsyncWebServer.h
